# ManjuWeb Agency Portfolio

Premium full-stack agency portfolio website built for modern web design excellence, responsiveness, and premium interactivity.

## 🚀 Key Features

*   **Modern Responsive Design**: Fluid grid layouts that adjust seamlessly across desktop, tablet, and mobile screens.
*   **Dark / Light Mode**: Dynamic, contrast-rich theme transitions with color tokens mapped directly to user system settings.
*   **AI Assistant**: Powered by a custom conversational sales pipeline that recommends tech stacks, generates website layout ideas, and estimates costs dynamically.
*   **Project Showcase**: Premium interactive browser frame mockups with zoom hover transitions and collapsible case studies.
*   **Hire Me / Client Intake**: Multi-step project configurator form with client-side anti-spam verification and rate limiting.
*   **Admin Dashboard Integration**: Full command center to monitor live telemetry metrics, review chatbot leads, publish blogs/testimonials, and configure services.
*   **Premium Animations**: Staggered fade-ins, spring micro-interactions, page loader transitions, and scroll animations powered by Framer Motion.

## 🛠️ Tech Stack

*   **Frontend**: React (v19) + Vite (v8) + Tailwind CSS (v4)
*   **Animations**: Framer Motion
*   **Icons**: Lucide React
*   **Tooling**: Concurrently (dev orchestration), Oxlint

## 💻 Local Development

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start development server**:
    ```bash
    npm run dev
    ```
3.  **Compile production bundle**:
    ```bash
    npm run build
    ```
